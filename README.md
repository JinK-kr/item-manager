# 우리 반 공용 물품 관리 앱

동아리방이나 실습실의 공용 물품을 여러 사람이 함께 등록하고 수량을 관리하는 앱입니다.
서버 없이 HTML 파일만으로 동작하고, 데이터는 브라우저에 저장됩니다.

## 화면

| 파일 | 역할 |
|---|---|
| `index.html` | 물품 등록 + 물품 목록 (수량 조절, 삭제) |
| `dashboard.html` | 현황 대시보드 (위젯 4개) |

## 기능

- **물품 등록** — 이름, 카테고리, 수량, 등록자 닉네임
- **수량 늘리기 / 줄이기** — `+` `−` 버튼. 0 아래로는 내려가지 않습니다
- **물품 삭제** — 확인 창을 거칩니다 (되돌릴 수 없음)

카테고리는 `문구류` · `전자기기` · `청소용품` · `기타` 네 가지로 고정입니다.

## 대시보드 위젯

1. **총 품목 수** — 등록된 물품 종류 수
2. **재고 부족 목록** — 수량 3개 이하인 물품
3. **카테고리별 분포** — 품목 수 기준 도넛 차트
4. **최근 변경 내역** — 수정 시간 최신순 5건

대시보드 아래 **테스트 도구**로 물품 10개를 한 번에 넣어 위젯이 어떻게 보이는지 확인할 수 있습니다.
`테스트 물품 지우기`는 그 버튼으로 넣은 물품만 지우고, 직접 등록한 물품은 건드리지 않습니다.

## 실행 방법

`index.html`을 더블클릭하면 열립니다.

브라우저가 `file://`에서 저장 기능(localStorage)을 막는 경우에는 화면 위에 경고 배너가 뜹니다.
그럴 때는 이 폴더에서 간단한 서버를 띄우세요.

```bash
python -m http.server 8765
```

그다음 `http://localhost:8765/index.html`로 접속합니다.

## 파일 구성

```
index.html / index.js        등록·목록 화면
dashboard.html / dashboard.js 대시보드 화면
app.js                        두 화면이 공유하는 데이터 계층 (localStorage)
style.css                     공용 스타일 (라이트/다크 자동 전환)
PRD.md                        제품 요구사항 문서
```

## 알아 둘 점

데이터는 **브라우저 localStorage**에 저장됩니다.
따라서 기기 한 대(브라우저 하나)에서 여러 사람이 번갈아 쓰는 구조입니다.
내 휴대폰에서 등록한 물품이 다른 사람 휴대폰에 자동으로 나타나지는 않습니다.
여러 기기에서 같은 목록을 보려면 서버나 별도 저장소가 필요하며, 이는 다음 버전 과제입니다.

## 이번 버전에서 만들지 않은 기능

로그인 · 검색 · 사진 업로드 · 대여와 반납 기능은 범위에서 제외했습니다.
자세한 배경과 결정 사항은 [PRD.md](PRD.md)에 정리돼 있습니다.
